## Writing Guidance
**Purpose:** Use input field text to guide users on how to enter the correct information.
<ul>
  <li><b>Avoid Punctuation:</b> Only use punctuation if you're posing a question.</li>
  <li><b>Use Sparingly:</b> Placeholder text should only be used when necessary. If the input field is self-explanatory, it may not need any additional hints.</li>
  <li><b>Avoid Redundancy:</b> Do not repeat the input field label in the placeholder.</li>
  <li><b>Consider Helper Text:</b> For more detailed guidance that remains visible as the user types, use persistent helper text instead of a disappearing placeholder.</li>
  <li><b>Case Sensitivity:</b> Always use sentence case for placeholder text.</li>
  <li><b>Standard Action Labels:</b> Use "Select" for dropdowns and other standard action labels for UI interactions within input fields.</li>
</ul>
