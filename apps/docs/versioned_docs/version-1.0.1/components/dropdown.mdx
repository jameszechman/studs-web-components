Dropdown components are used for collecting user provided information from a list of options.

## Usage

<Tables
  hideHeader
  headers={['title', 'description']}
  args={[
    {
      title: 'Usage and conditions (for acceptance criteria)',
      description: (
        <div>
          <p>
            Use the dropdown component to filter, sort, or change page content.
            Unlike the select component, dropdown can be styled to match the
            look and feel of a product or application. Dropdowns use multiple
            concurrent selections, filtering, and grouping to accommodate long
            lists.
          </p>
          <b>Anatomy</b>
          <div>
            The dropdown component is composed of two main parts: a prompt and a
            menu option list. The component is customizable so that it can
            accommodate multiple design objectives. Customizations can affect
            the dropdown’s anatomy and appearance depending on its intended use.
            The following example shows both parts of a single-select dropdown
            with a simple list and all possible elements.
          </div>
          <div>The dropdown prompt consists of four primary elements:</div>
          <ol>
            <li>Label: describes the options the user will see</li>
            <li>
              Dropdown field: an empty field that indicates a user needs to
              select an option from the list. This field can be modified to
              include a prompt so that the user can type in it
            </li>
            <li>Helper text (optional): message to help or guide the user</li>
            <li>Chevron: visual indication that the list can open or close</li>
          </ol>
          <div>
            When the user expands the dropdown, the options are visible:
          </div>
          <ol>
            <li>
              Menu option list: shows a user their options. This part is
              customizable as a simple, grouped, or multi-select list
            </li>
            <li>
              Scrollbar: used only when the quantity of items in the menu option
              list exceeds the height of the viewport
            </li>
            <li>Check icon: visual representation of one selected option</li>
          </ol>
        </div>
      ),
    },

    {
      title: 'Style',
      description: (
        <div>
          <b>Single-select</b>
          <p>
            Use the single-select dropdown when selecting one value from a list.
            The selected option then appears in the field.
          </p>
          <br />
          <b>Multi-select</b>
          <p>
            The multi-select dropdown allows the user to select more than one
            value from a list. The number (quantity) of selected options appears
            as a tag in the field.
          </p>
          <br />
          <p>
            Use a “Select all” checkbox with a multi-select list containing
            larger lists of options. The “Select all” checkbox is not visible
            while a&nbsp;
            <a
              href="https://www.delldesignsystem.com/patterns/filter/"
              target="_blank"
            >
            filter
            </a>
            &nbsp;term is being entered into the filtering text field.
          </p>
          <br />
          <b>With filtering</b>
          <p>
            You can enable filtering on any of the dropdown lists. When
            filtering is enabled, the dropdown field functions as a text input
            in which the user can type. The results are alphabetized and reflect
            any of the letters that appear in the word.
          </p>
          <p>
            If none of the results reflect what the user typed, a “No options
            found” message will appear in the dropdown list.
          </p>
          <br />
          <b>Kinds</b>
          <p>The menu of the dropdown can accommodate three kinds of lists.</p>
          <br />
          <b>Simple list</b>
          <p>A dropdown with a simple list is a menu with a set of options.</p>
          <br />
          <b>Grouped list</b>
          <p>
            A dropdown with a grouped list allows options to be grouped by
            category. Grouped lists can function either as simple or as
            multi-select lists. Note that the grouping titles cannot be
            selected.
          </p>
          <br />
          <b>Multi-select list</b>
          <p>
            A dropdown with a multi-select list includes checkboxes to indicate
            that the user can select any number of options. Add a “Select all”
            checkbox to select or deselect all options in the list.
          </p>
          <br />
          <b>Grouped multi-select list</b>
          <p>
            A dropdown with options grouped by category and checkboxes to
            indicate that the user can select any number of options. A “Select
            all” checkbox allows a user to select all options in every group.
            Deselecting the “Select all” checkbox clears all selections in all
            groups.
          </p>
          <br />
        </div>
      ),
    },

  ]}
