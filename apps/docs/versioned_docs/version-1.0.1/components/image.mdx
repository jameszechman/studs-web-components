## Writing Guidelines
<p> When writing alternative text, use specific, concise language that conveys the essential information about the image to support screen reader users. Limit text to 125 characters or less.</p>
<div>
  <h3>Avoid Redundant Phrasing</h3>
  <p>Describe the image content without stating that it is an image, as the context is already clear.</p>
  <div class='exampleImages'>
    <div class='wrapper -do'>
      <p class='-do'>Do</p>
      <figure>
        <img src='/img/Content_Guidance_Images/Image/DO_Image_Redundant.png' alt='Example image of construction that uses alternative text "Rebar placement in concrete footing for load-bearing walls.".' />
        <figcaption></figcaption>
      </figure>
    </div>


    <div class='wrapper -dont'>
      <p class='-dont'>Don't</p>
      <figure>
        <img src='/img/Content_Guidance_Images/Image/DONT_Image_Redundant.png' alt='Example of image of construction that uses alternative text "Image of construction."' />
        <figcaption></figcaption>
      </figure>
    </div>
  </div>



  <h3>Show Instead of Telling</h3>
  <p>Use specific, concise language that conveys the essential information about the image, supporting screen reader users and those who cannot see the image.</p>
  <div class='exampleImages'>
    <div class='wrapper -do'>
