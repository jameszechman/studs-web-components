# Usage
## How to use STUDS Components

<studs-tabs tabdirection="row">
  <button slot="tab">Vanilla JS \ HTML</button>
  <section slot="panel">

  #### Import the CSS
  ```js
  import "@studs/ui/studs-base.css"
  ```
  #### Import Components
  This will import all the components available in Studs, if you want to import only certain element, see below to import individual components.
  ```js
  import "@studs/ui"
  ```
  *or*
  #### Import Individual Components
  ```js
  import "@studs/ui/display/button"
  ```
  #### Use a component
  ```html
  <studs-button button-type="primary">Button</studs-button>
  ```

  </section>

  <button slot="tab">React</button>
  <section slot="panel">

  #### Import the CSS
  ```js
  import "@studs/react/studs-base.css"
  ```
  #### Import Component
  ```jsx
  import { StudsButton } from "@studs/react"
  ```
  #### Use a component
  ```jsx
    <StudsButton button-type="primary">Button</StudsButton>
  ```

  </section>

  <button slot="tab">Next</button>
  <section slot="panel">

    We are currently working on SSR within Studs. Until then, next requires a different setup to begin using Studs.
    #### Import the CSS
    ```js
    import "@studs/react/studs-base.css"
    ```
    #### Import Component
    ```jsx
      import dynamic from 'next/dynamic';
      const StudsButton = dynamic(() => import('@studs/react').then((mod) => mod.StudsButton), { ssr: false });
    ```
    #### Use a component
    ```jsx
    <StudsButton button-type="primary">Button</StudsButton>
    ```

  </section>
