# Installation

Install only one of the packages. `@studs/ui` or `@studs/react`.

<studs-tabs tabdirection="row">
  <button slot="tab">Vanilla JS \ HTML</button>
  <section slot="panel">

  #### npm
  ```zsh
  npm install @studs/ui
  ```
  #### yarn
  ```zsh
  yarn add @studs/ui
  ```
  #### pnpm
  ```zsh
  pnpm add @studs/ui
  ```

  </section>

  <button slot="tab">React</button>
  <section slot="panel">

  #### npm
  ```zsh
  npm install @studs/react
  ```
  #### yarn
  ```zsh
  yarn add @studs/react
  ```
  #### pnpm
  ```zsh
  pnpm add @studs/react
  ```

  </section>
</studs-tabs>
